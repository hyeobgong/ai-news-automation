## 1. 사용자의 핵심 요청 및 비전
1.  **서비스 확장:** 단순 개인/팀 리포트 생성에서 **블로그, 인스타그램, 유튜브, 커뮤니티 등 멀티 채널 자동 업로드**로 확장 희망.
2.  **뉴스 데이터 확대:** 현재 20개 제한을 넘어, 더 많은 분류와 수량을 확보하길 원함.
3.  **유튜브 특화:** 영상의 경우 소수의 중요 뉴스(10~20개) 집중 조명.
4.  **비용 효율성:** 무료에 가까운 운영 방식 확인 완료 (Gemini Flash 및 GitHub Actions 활용 시 비용 0원 유지 가능).

---

## 2. 4단계 확장 로드맵 (Action Plan)

### [Phase 1] DB 구축 및 분류 체계 고도화 (Base)
- **현재 문제:** 뉴스를 20개만 수집하고 나머지는 버림. 데이터가 휘발됨.
- **해결 방안:**
    - SQLite 또는 JSON 파일 시스템 도입하여 수집된 **모든 뉴스** 저장.
    - 카테고리 태깅 세분화 (LLM / Agent / Hardware / Business / Ethics).
    - "오늘의 뉴스" 뿐만 아니라 주간/월간 리포트 생성 기반 마련.

### [Phase 2] 블로그/텍스트 콘텐츠 자동 발행 (Text)
- **목표:** 티스토리, 워드프레스, 네이버 블로그 등 텍스트 기반 매체.
- **구현 기능:** Markdown 리포트를 **HTML 포맷**으로 자동 변환, API를 통한 자동 업로드.

### [Phase 3] SNS용 이미지 자동 생성 (Image)
- **목표:** 인스타그램, X(트위터), 페이스북 등 시각적 매체.
- **구현 기능:**
    - Python `Pillow` 라이브러리로 **카드뉴스 이미지** 자동 생성.
    - 썸네일 + 3줄 요약 텍스트 합성.

### [Phase 4] 유튜브 쇼츠 영상 자동화 (Video)
- **목표:** 유튜브 쇼츠, 틱톡, 릴스 등 숏폼 영상.
- **구현 기능:**
    - `MoviePy` 라이브러리로 이미지 슬라이드 영상 제작.
    - TTS (Text-to-Speech)로 요약문 읽어주는 음성 추가.
    - 배경 음악 및 자막 자동 삽입.

## 3. 주요 Q&A 기록

### Q1. 비용은 얼마나 드나요?
**A:** 개인 규모에서는 **사실상 0원**입니다.
- **AI API:** Google Gemini Flash 모델의 Free Tier 사용 시 하루 1,500회 무료 (현재 20~30회 사용).
- **서버:** 로컬 PC 사용 시 0원, GitHub Actions 사용 시 월 2,000분 무료 (충분함).

### Q2. 한국 뉴스와 해외 뉴스 비중 문제?
**A:** 초기에는 수집 순서 때문에 한국 뉴스만 선택되었으나, **미국 뉴스 우선 수집 및 2:1 비율 혼합** 로직을 적용하여 해결했습니다. (`main.py` 수정 완료)

### Q3. 영문 기사 번역은?
**A:** `Gemini` 프롬프트 수정을 통해 제목과 요약문을 **전부 한국어로 번역**하여 출력하도록 기능 추가 완료.
